##  MDE Sense Event IDs

These events are generated by the **MsSense.exe** service, which powers the behavioral sensor for MDE.

###  Event ID Reference Table

| **Event ID** | **Message** | **Description** | **Action** |
|--------------|-------------|------------------|------------|
| `1` | Service started | Occurs during startup, shutdown, or onboarding | No action needed |
| `2` | Service shutdown | Triggered during shutdown or offboarding | No action needed |
| `3` | Service failed to start | Includes failure code | Investigate cause and logs |
| `4` | Connected to server | Shows URL of MDE processing server | Informational |
| `5` | Failed to connect to server | Couldn’t reach MDE cloud | Check proxy/network settings |
| `6` | Not onboarded | No onboarding parameters found | Re-run onboarding script |
